# 🧠 Multi-Agent Conversational Assistant using LangChain

## Overview

This project demonstrates the creation of multiple conversational AI agents using LangChain and Gemini.

Each agent specializes in a different domain and maintains conversation memory across interactions.

---

## Agents Included

### General Assistant Agent

Capabilities:

- Answer questions
- Maintain context
- Use custom tools
- Multi-turn conversations

---

### HR Interview Agent

Features:

- Conducts mock interviews
- Evaluates candidate responses
- Generates follow-up questions
- Simulates real HR rounds

---

### Doctor Consultation Agent

Features:

- Collects patient information
- Understands symptoms
- Maintains conversation history
- Provides structured medical guidance

---

### Placement Preparation Agent

Features:

- Conducts placement interviews
- Generates technical questions
- Provides interview preparation guidance
- Simulates campus recruitment rounds

---

## Features
